W Web Launcher

Customize homescreen with your web skill

How to change website

  • Create website and save as index.html
  • You can use other assets such as: images, CSS, JavaScript in same folder
  • You can open an app installed in your phone by call js function openApp('packageName'), for example: openApp('com.android.dialer')
  • Open menu settings in corner top right and choose folder contain file index.html
